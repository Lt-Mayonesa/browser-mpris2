hackAudioTags()

Here we replace the document.createElement function with our own so we can detect when an <audio> tag is created that is not added to the DOM which most pages do while a <video> tag typically ends up being displayed to the user, audio is not.

Original code from https://github.com/KDE/plasma-browser-integration/blob/master/extension/content-script.js
